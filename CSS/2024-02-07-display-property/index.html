CSS Display Özelliği

Etiketlerin display özellik değerleri default oalrak aşağıdaki değerlerden biri olabilir.

  • display:inline
  • display:block
  • display:inline-block
  • display:none

  • display:flex
  • display:grid

Inline Elements

  • inline öğlere sator okuma yönünde hizalanır.
    • width ve height verilemez. bu etiketler, alan etiketleri olarak düşünülmemiştir.
  • default genişlikleri auto'dur. içeriklerine göre otomatik değişir.
  • yükseklikleri, satır yüksekliğine bağlıdır. (line height)
  • inline öğeler, kapsayıcılarından birine verilen text-align özelliğinin left, right, center, justify değerleri kullanılarak okuma yönünde hizalanabilir.
  • vertical-align özelliğinin top, midle, bottom değerleri ile satır içerisinde dikeyde hizalanabilirler.

Block Elements

  • block öğelerin öynü dikey yöndür. block öğeler alt alta dizilir.
  • default genişlikleri 100%
  • width ve height verilebilir.
  • block öğelerden kalan pozitif boşluk, pozitive space, margin sağ ve sol değeri auto yapılarak paylaştırılabilir.
